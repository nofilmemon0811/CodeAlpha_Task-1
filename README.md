Iris Classification using Logistic Regression
Project Overview

This project is part of my Internship Task 1, where I built a Machine Learning classification model using the famous Iris dataset.
The goal was to classify flower species using Logistic Regression after preprocessing the data.

Dataset
Dataset used: Iris Dataset (Seaborn)
Classes:
Setosa (removed)
Versicolor
Virginica
Steps Performed
Data Preprocessing
Loaded dataset using Seaborn
Removed Setosa class to simplify classification
Converted categorical labels into numerical:
Versicolor → 0
Virginica → 1
Model Building
Split data into training and testing sets
Applied Logistic Regression model from Scikit-learn
Model Evaluation
Evaluated model using:
 Accuracy Score
 Confusion Matrix
 Classification Report
 Prediction
Implemented user input prediction to test the model with custom values
